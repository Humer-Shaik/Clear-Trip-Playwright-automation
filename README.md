# Clear-Trip-Playwright-automation

Clear-Trip-Playwright-automation is a focused end-to-end test suite that automates common user journeys on the Cleartrip website using Playwright with Python bindings. The repository contains modular test code, page models, and HTML artifacts intended to make UI test coverage expressive, maintainable, and easy to extend.

Key highlights:

- Automated browser-based tests for search, filtering, and booking-related UI flows.
- Organized test structure and reusable page objects to keep scenarios readable and maintainable.
- Test artifacts and HTML reports to aid troubleshooting and review of failures.
- Written primarily in Python with supporting HTML report files.

This project is intended as a practical example of using Playwright to validate critical web interactions, regressions, and end-to-end behaviors on a travel booking site.
